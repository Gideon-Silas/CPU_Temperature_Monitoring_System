# CPU Temperature Monitoring System

In this project, I analyzed CPU temperature readings using Python. The objective was to apply fundamental Python programming concepts while working with immutable data stored in a tuple. The project focuses on retrieving temperature values, performing statistical calculations, and understanding when tuples are more appropriate than lists for storing fixed datasets.

##### The dataset consists of five CPU temperature readings recorded during system operation:

45, 50, 55, 48, 52

Using this dataset, I performed a series of operations to retrieve specific temperature readings, calculate key statistics, and evaluate why a tuple is a suitable data structure for storing CPU temperature records. This project demonstrates how Python tuples and built-in functions simplify data analysis while protecting data from unintended modification.

#### The objectives of this project were to:

- Store CPU temperature readings in a Python tuple.
- Retrieve the first temperature reading.
- Retrieve the last temperature reading.
- Calculate the difference between the highest and lowest temperatures.
- Calculate the average CPU temperature.
- Explain why a tuple is more suitable than a list for storing CPU temperature data.

#### This project strengthened my understanding of:

- Creating and using Python tuples.
- Accessing elements through indexing.
- Retrieving the first and last values in a dataset.
- Performing statistical calculations using built-in Python functions.
- Understanding the difference between mutable and immutable data structures.
- Writing clean and readable Python code.

#### By completing this project, I learned how to:

- Store fixed datasets using Python tuples.
- Retrieve specific values using indexing.
- Calculate summary statistics from numerical data.
- Analyze CPU temperature readings using Python.
- Identify situations where tuples are more appropriate than lists.

This project introduces the use of tuples for storing fixed datasets in Python while performing basic data analysis on CPU temperature readings.
